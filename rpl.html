Jurusan RPL (Rekayasa Perangkat Lunak)

1. Apa itu RPL?

RPL atau Rekayasa Perangkat Lunak adalah jurusan yang mempelajari cara merancang, membuat, mengembangkan, dan memelihara perangkat lunak (software). Jurusan ini sangat cocok untuk siswa yang tertarik pada teknologi, logika, dan pemrograman komputer.

2. Apa yang Dipelajari di Jurusan RPL?

  • Pemrograman: Bahasa seperti Python, Java, C++, JavaScript.
  • Basis Data: MySQL, PostgreSQL, MongoDB.
  • Pengembangan Web: HTML, CSS, JavaScript, PHP, Laravel.
  • Mobile App Development: Android Studio, Kotlin, Flutter.
  • Analisis & Perancangan Sistem: UML, DFD, ERD.
  • Testing & Debugging: Unit testing, error handling.
  • Manajemen Proyek: Metode Agile, Scrum, Git & GitHub.

3. Jenjang Karier Lulusan RPL

  • Software Developer / Programmer
  • Mobile App Developer
  • Web Developer / Full Stack Developer
  • Database Administrator
  • System Analyst
  • Quality Assurance / Software Tester
  • IT Support / Technical Support
  • Freelancer atau Wirausaha di bidang Teknologi
